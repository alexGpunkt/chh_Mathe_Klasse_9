# Entwicklermenü

Das Käfer-Symbol unten links erscheint nur bei `devMode: true` in `assets/js/supabase-config.js`.

Funktionen:
- Schülerlogin überspringen
- Tracker deaktivieren
- Supabase deaktivieren
- drei getrennte Testschüler wählen
- Testevent und Testfortschritt erzeugen
- Offlinecache löschen
- Service Worker aktualisieren
- Dashboard öffnen
- echten Login gezielt testen

Vor dem Merge nach `master`:

```js
devMode: false
```
